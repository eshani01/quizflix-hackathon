A streaming-inspired gamified learning platform. We used streaming-platform inspired UI to make learning addictive and familiar like anyother series we binge watch. 

What This Project Does :
	•	Allows a user to log in (simulated for now).
	•	Lets a student enter a class code to access course content.
	•	Displays subjects in a streaming-style dashboard.
	•	Opens a lesson preview before starting the quiz.
	•	Treats questions as “episodes”.
	•	Tracks performance using timers and streaks.

Key Features of this Project :
1. Landing & Login Flow
	•	Clean landing page with branding.
	•	Simple login flow (frontend-based).
	•	Teacher class code system

2. Course Dashboard
	•	Subjects displayed like streaming profiles.
	•	Blurred background styling.
	•	Students cannot add or delete courses.
	•	Avatar editing available in edit mode.

3. Lesson Preview Modal
	•	Clicking a lesson opens a centered preview screen.
	•	Background blurs while the preview is active.
	•	Shows:
	•	Lesson title
	•	Overview
	•	Episode list (questions)
	•	Play button

4. Episode-Based Quiz System
	•	Questions are labeled as “episodes”.
	•	Only one option can be selected per question.
	•	Selected answers are visually highlighted.
	•	Correct answers are shown clearly.
	•	“Next Episode” button moves to the next question.

5. Timer & Performance Tracking
	•	Each question has its own timer.
	•	Time taken per question is recorded.
	•	Total time is calculated.
	•	Correct answers are tracked.
	•	Consecutive correct answers trigger Binge Mode.

6. Binge Mode
	•	Activated after 3 correct answers in a row.
	•	Displays a popup notification.
